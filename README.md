# n8n Automations

A collection of automation workflows built with n8n, including email response systems powered by AI.

## 📋 Workflows

### 1. A Very Simple Human in the Loop Email Response System Using AI and IMAP
An email automation system that integrates with IMAP to read emails and uses AI to help generate responses, with human approval workflows.

### 2. AI Email Auto-Responder with Ollama
An advanced email auto-responder system that uses Ollama (local AI) to automatically generate and send email responses.

## 🚀 Getting Started

1. Install [n8n](https://n8n.io/)
2. Import the JSON workflow files into n8n
3. Configure your email credentials (IMAP settings)
4. Set up your AI model (Ollama or cloud-based)
5. Deploy and run your workflows

## 📦 Requirements

- n8n instance (self-hosted or cloud)
- Email account with IMAP access
- AI model (Ollama for local setup or cloud alternatives)
